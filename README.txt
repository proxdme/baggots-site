BAGGOTS SITE — первая версия сайта-визитки

Как открыть:
1. Распакуй архив.
2. Открой файл index.html двойным кликом.

Что внутри:
- index.html — структура сайта.
- styles.css — внешний вид.
- script.js — карточки сумок, тексты, ссылки и логика открытия карточек.
- images/ — фотографии и заглушки.

Как заменить фото:
1. Зайди в папку images.
2. Замени файл нужной модели, например:
   red-zip-1.png — главное фото Red Zip
   red-zip-2.svg — второе фото, можно заменить на red-zip-2.jpg или red-zip-2.png
3. Если меняешь расширение файла, не забудь поправить путь в script.js.

Как изменить цену, статус, описание:
1. Открой script.js.
2. Найди нужную сумку по id или title.
3. Измени поля price, status, description, materials, size.

Где ссылки:
- VK уже прописан: https://vk.com/baggots
- Instagram уже прописан: https://www.instagram.com/baggots_shop/
- MAX и Threads стоят заглушками #. Когда появятся ссылки, замени # на реальные адреса.

Как выложить бесплатно:
Самый простой способ — Netlify:
1. Зайди на https://www.netlify.com/
2. Зарегистрируйся.
3. Перетащи папку baggots-site в deploy-зону.
4. Получишь ссылку на сайт.

Важно:
Это первая рабочая версия. Она не требует базы данных, сервера и CMS.


ЛОГОТИП
-------
В папке images/brand лежат исходники логотипа:
- logo-header.svg — основной логотип в шапке сайта;
- logo-full.png — полный PNG-вариант;
- logo-avatar.png — круглая/квадратная аватарка с козликом;
- logo-original.jpg — исходный JPG.

Как заменить логотип:
1. Если новый логотип в SVG — назови его logo-header.svg и замени файл images/brand/logo-header.svg.
2. Если хочешь использовать PNG/JPG, открой index.html и в строке с class="brand-logo" поменяй src на нужный файл, например images/brand/logo-full.png.
3. Для аватарки и иконки сайта меняется файл images/brand/logo-avatar.png.


Обновление v3:
- Добавлен двуязычный блок описания: RU / EN.
- Шрифт заменён на более мягкий системный стек: Trebuchet MS / Segoe UI / Arial.
- Уменьшена агрессивная плотность заголовков.
- В карточках указан примерный размер 300 × 150 × 70 мм.

Как поменять шрифт:
Открой styles.css и найди строку body { ... font-family: ... }.
Можно попробовать варианты:
1) "Trebuchet MS", "Segoe UI", Arial, sans-serif — текущий, мягкий и дружелюбный.
2) "Segoe UI", Arial, sans-serif — более нейтральный.
3) Georgia, "Times New Roman", serif — более журнальный, но менее технологичный.
